* Key bindings

| Key Binding   | Description   |
|---------------+---------------|
| ~<SPC> a s r~ | Start Spray   |
| ~h~           | Backward word |
| ~l~           | Forward word  |
| ~f~           | Faster speed  |
| ~s~           | Slower speed  |
| ~q~           | Quit Spray    |
